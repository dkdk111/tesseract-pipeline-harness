# Task: competitive market brief

## Goal

Produce a competitive market brief on the note-taking app space, covering the main
players on pricing, features, and positioning, ending with a short recommendation.

## Input

- Scope: three primary competitors (Notion, Obsidian, Roam).
- Dimensions to cover: pricing, features, positioning.
- Output: one brief document, plus a one-paragraph recommendation.

## Box in force for this run

Read from `harness/box.config.md`: all four axes allowed, max_depth 3,
max_breadth 6, max_rounds 3. No approval-required actions are involved (nothing
leaves the repo, nothing irreversible).

## Note

This file is what a human hands the harness. What the agent produced from it, as a
self-design trace, is in `trace.md` and `tesseract.json`. The point of the example
is not the brief itself but the shape the harness gave it: one ordinary task, all
four axes.
